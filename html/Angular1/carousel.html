mui与angularjs实现轮播刷新数据

作者：吴杨杨 返回首页

时间：2017-01-11

要实现图片轮播的效果，只需将Hello mui中的代码示例拷贝就可以，js代码如下：

var slider = mui("#slider");
slider.slider({
   interval: 0 // 自动轮播周期，若为0则不自动播放，默认为0
});

4、如果需要根据切换的图片刷新数据，使用如下代码：

document.querySelector('.mui-slider').addEventListener('slide', function(event) {
   // slideNumber是从0开始的;
   var page = event.detail.slideNumber;

   $scope.$apply(function() {
      $scope.data = $scope.dataList[page];
   });
});

$apply接收表达式或者函数作为参数后调用$digest来更新绑定以及监控器。实际上，Angular几乎在所有提供的代码中添加了$apply，如ng-click，初始controller，$http的回调操作，在这，你并不需要亲自调用 $apply，而且重复的调用会引起错误。
