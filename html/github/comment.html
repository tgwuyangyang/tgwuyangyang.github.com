使用github中的Issues给博客搭建评论功能

作者：吴杨杨 返回首页

时间：2017-06-16

过年的时候本来就要给博客搭建一个评论功能，由于多说的关闭，再加上网易之类的不支持https，所以一直拖到现在。

我的博客搭建留言功能使用的是github中的Issues,是github上一个大神做出来的功能。项目地址

使用方法如下：

1.注册 OAuth Application，点击此处来注册一个新的 OAuth Application。其他内容可以随意填写，但要确保填入正确的 callback URL（一般是评论页面对应的域名，如 https://imsun.net）。 你会得到一个 client ID 和一个 client secret，这个将被用于之后的用户登录。

2.引入 Gitment，将下面的代码添加到你的页面：

<div id="container"></div>
<link rel="stylesheet" href="https://imsun.github.io/gitment/style/default.css">
<script src="https://imsun.github.io/gitment/dist/gitment.browser.js"></script>
<script>
var gitment = new Gitment({
  id: '页面 ID', // 可选。默认为 location.href
  owner: '你的 GitHub ID',
  repo: '存储评论的 repo',
  oauth: {
    client_id: '你的 client ID',
    client_secret: '你的 client secret',
  },
})
gitment.render('container')
</script>

注意，上述代码引用的 Gitment 将会随着开发变动。如果你希望始终使用最新的界面与特性即可引入上述代码。

提醒使用本项目的用户，请保持克制，切勿滥用。

更多使用方法可以具体查看我博客的源码博客源码

如果不喜欢这种方式搭建博客，还可以使用Comm(ent|it)搭建，也是开源的。搭建方法
