session共享

作者：吴杨杨 返回首页

时间：2016-10-15

搭建nginx集群时，还需要考虑一个重要问题--session共享

1.最简单的方法就是通过ip分配，nginx.conf文件中配置ip_hash,如下：

upstream myserver {
   ip_hash;
   server 192.168.0.12:80;
   server 192.168.0.13:80;
}

nginx中的ip_hash技术能够将某个ip的请求定向到同一台后端，这样一来这个ip下的某个客户端和某个后端就能建立起稳固的session，缺点就是某台服务器宕掉后这台服务器上的用户照样无法访问
