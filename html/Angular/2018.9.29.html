angular6环境搭建

作者：吴杨杨 返回首页

时间：2018-09-29

由于最近要用到angular6，所以好好学习一番。

首先，从搭建环境做起。

搭建本地开发环境前，需要安装node和npm。请先在终端/控制台窗口中运行命令 node -v 和 npm -v， 来验证一下你正在运行 node 8.x 和 npm 5.x 以上的版本。 更老的版本可能会出现错误，更新的版本则没问题。

由于以前安装的node版本过低，所以安装了最新版本，执行npm -v命令查看npm版本时，卡住了，没有结果，也不报错。

处理方法，删除C:\Users\{账户}\下的.npmrc文件。

再运行npm命令，正常。

确认你运行的 node 是8.x或更高，npm 为5.x或更高，进行以下操作：

1.设置开发环境,全局安装 Angular CLI：

  • npm install -g @angular/cli

2.创建新项目

  • ng new my-app

执行到这一步时，下载资源的时候又卡住了，于是将npm的镜像替换成淘宝。

  • npm config set registry http://registry.npm.taobao.org/

删除掉my-app文件夹后，再重新执行ng new my-app就好了。

3.启动开发服务器

  • cd my-app
  • ng serve --open

使用 --open（或 -o）参数可以自动打开浏览器并访问 http://localhost:4200/。

总结以下，其实就3个命令

  • npm install -g @angular/cli
  • ng new my-app
  • ng serve --open

以上。
