angular6路由常用属性

作者：吴杨杨 返回首页

时间：2019-05-09

使用Angular6开发App时，footer菜单在使用路由跳转后，左滑或者点击返回键不能返回。

所以得使用到NavigationExtras中的replaceUrl属性。

  • this.router.navigate([path], { replaceUrl: true });

url传递参数

使用NavigationExtras中的queryParams属性

  • this.router.navigate([path], { queryParams: {id: '1'} });

即要url传递参数，也要删除历史页面。

  • this.router.navigate([path], { queryParams: {id: '1'}, replaceUrl: true });
