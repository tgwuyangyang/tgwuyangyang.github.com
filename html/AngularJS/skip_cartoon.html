angularjs学习笔记--SPA单页面应用页面跳转类似于微信的动画效果

作者：吴杨杨 返回首页

时间：2016-09-21

进入二级页面时，页面从右向左滑动进入。html代码如下：

<div ng-view="" class="slide-left"></div>

css代码如下：

.slide-left {
   position: absolute;
   left: 0;
   top: 0;
   width: 100%;
   height: 100%;
}

.slide-left.ng-enter, .slide-left.ng-leave {
   -webkit-transition: all 0.5s ease;
   transition: all 0.5s ease;
}

.slide-left.ng-enter {
   left: 100%;
}

.slide-left.ng-enter-active {
   left: 0;
}

.slide-left.ng-leave {
   left: 0;
}

.slide-left.ng-leave-active {
   left: -100%;
}

返回上级页面时，页面从左向右滑动进入。html代码如下：

<div ng-view="" class="slide-right"></div>

css代码如下：

.slide-right {
   position: absolute;
   left: 0;
   top: 0;
   width: 100%;
   height: 100%;
}

.slide-right.ng-enter, .slide-right.ng-leave {
   -webkit-transition: all 0.3s ease;
   transition: all 0.3s ease;
}

.slide-right.ng-enter {
   left: -100%;
}

.slide-right.ng-enter-active {
   left: 0;
}

.slide-right.ng-leave {
   left: 0;
}

.slide-right.ng-leave-active {
   left: 100%;
}
