box-sizing

作者：吴杨杨 返回首页

时间：2016-08-20

下面这段css很简单，就是设置一个div的宽为200px，高为50px

.div {
   width: 200px;
   height: 50px;
}

可当我们给它加上内边距或者边框时

.div {
   width: 200px;
   height: 50px;
   padding: 20px;
   border solid 10px #000000;
}

div元素看起来没什么不同，but

实际展现的div却能够超出你的设置，因为元素的边框和内边距会撑开元素

以前，我的解决方法是数学加减，现在我不用这么辛苦了

css有一个属性叫box-sizing，当你设置一个元素为box-sizing: border-box;时，此元素的内边距和边框不再会增加它的宽度

.div {
   width: 200px;
   height: 50px;
   padding: 20px;
   border solid 10px #000000;
   -webkit-box-sizing: border-box;
   -moz-box-sizing: border-box;
   box-sizing: border-box;
}

如果你没有比这更好的办法，就把这些css放在你的页面上吧。

.div {
   -webkit-box-sizing: border-box;
   -moz-box-sizing: border-box;
   box-sizing: border-box;
}
